Cómo se usa:

El primer uso:


./WebApp -t "Título de la aplicación" -u "URL" -w "Ancho de la aplicación" -H "Altura de la aplicación"

El segundo uso:


Use "WebApp -h" para obtener ayuda

Use: ./WebApp [opciones] Description: WebApp opciones: -h, --help Muestra este mensaje de ayuda. -v, --version Mostrar información de la versión. -p, --parser Habilite el CommandLineParser. El valor por defecto es falso. -t, --title <título> El título de la aplicación Por defecto es WebApp. -u, --url <url> La URL del objetivo. Por defecto está en blanco. -w, --width <ancho> El ancho de la aplicación. Por defecto es 1024. -H, --height <altura> La altura de la aplicación. Por defecto es 768. -i, --ico <icono> El Icono de la aplicación. -d, --desc <descripción>La descripción de la aplicación. -c, --cfg <cfg> El archivo de configuración de la aplicación. -r, --root <root> La ruta raíz del servicio web del programa. -P, --port <puerto> El número de puerto del servicio web del programa.
Cabe señalar que la línea de comandos sobrescribirá los valores del archivo de configuración.

How to Usage:

The first usage:


WebApp ApplicationTitle TargetURL ApplicationWidth ApplicationHeight

The second usage:


Using "WebApp -h" to get help

Usage: ./WebApp [options] Description: WebApp Options: -h, --help Show this help Message. -v, --version Show Version info. -p, --parser Enable CommandLineParser. Default is false. -t, --title <title> The Title of Application. Default is WebApp. -u, --url <url> The target URL. Default is Blank. -w, --width <width> The Width of Application. Default is 1024. -H, --height <height> The Height of Application. Default is 768 -i, --ico <ico> The ICON of Application. -d, --desc <desc> The Description of Application. -c, --cfg <cfg> The Configuration file of Application. -r, --root <root> The root path of the program web service. -P, --port <port> The port number of the program web service.
It should be noted that the command line will overwrite the configuration information of the configuration file.

帮助:

第一种用法:


WebApp 程序标题 程序访问网址 运行宽度 运行高度

第二种用法:


运行 "WebApp -h" 查看命令行帮助信息

用法：./WebApp [选项] 描述: WebApp 选项： -h, --help 显示这个此帮助。 -v, --version 显示版本信息。 -p, --parser 启用参数解析方式. 默认顺序解析方式. -t, --title <title> 设置程序的运行标题. 默认为WebApp. -u, --url <url> 设置要打开的目标URL. 默认是空. -w, --width <width> 设置程序运行的宽度. 默认是 1024. -H, --height <height> 设置程序运行的高度. 默认是 768 -i, --ico <ico> 设置程序运行的图标. -d, --desc <desc> 设置程序的描述信息. -c, --cfg <cfg> 设置程序运行的配置文件. -r, --root <root> 设置内置WebServer的根路径. -P, --port <port> 设置内置WebServer的监听端口号.
需要注意的是, 命令行会覆盖配置文件的配置信息.
